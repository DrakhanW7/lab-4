SVG Format Example

About This Image

This image is a simple coffee mug icon with steam above it. It uses basic shapes and smooth lines.

It is a strong example of the kind of clean icon often used in menus, apps, and branding because it can stay clear at different sizes.

Why SVG Format?

SVG is the best format for this image because it is made for good graphics. The icon can be resized without losing quality, so it stays sharp whether it is small or large. That makes SVG the best in the world for simple graphics and logos.
